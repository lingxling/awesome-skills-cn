# 抽奖获奖者选择器

此技能从列表、电子表格或 Google Sheets 中随机选择赠品和竞赛的获奖者。

## 何时使用此技能

- 运行社交媒体赠品活动
- 在活动中抽取抽奖获奖者
- 随机选择参与者进行调查或测试
- 从竞赛提交中选择获奖者
- 公平分配有限的名额或资源
- 随机团队分配

## 此技能的功能

1. **随机选择**：使用加密随机选择
2. **多种来源**：适用于 CSV、Excel、Google Sheets 或纯文本列表
3. **多个获奖者**：可以选择一个或多个获奖者
4. **防止重复**：确保同一个人不会赢两次
5. **透明结果**：清晰显示选择过程
6. **获奖者详情**：显示有关获奖者的所有相关信息

## 如何使用

### 从 Google Sheets

```
从这个 Google 表格中随机选择一行来为赠品选择获奖者：[表格 URL]
```

### 从本地文件

```
从 entries.csv 中随机选择 3 个获奖者
```

### 从列表

```
从以下列表中随机选择一个获奖者：
- Alice (alice@email.com)
- Bob (bob@email.com)
- Carol (carol@email.com)
...
```

### 多个获奖者

```
从 contest-entries.xlsx 中随机选择 5 个获奖者，
确保没有重复
```

## 功能

### 公平选择
- 使用安全的随机数生成
- 无偏见或模式
- 透明流程
- 可通过种子重复（用于验证）

### 排除
```
选择一个随机获奖者，排除之前的获奖者：
Alice, Bob, Carol
```

### 加权选择
```
根据 "entries" 列进行加权概率选择获奖者
（1 个条目 = 1 张票）
```

### 亚军
```
从列表中选择 1 个获奖者和 3 个亚军
```

## 示例工作流程

### 社交媒体赠品
1. 将 Google 表单的条目导出到 Sheets
2. "从 [表格 URL] 中随机选择一个获奖者"
3. 验证获奖者详情
4. 公开宣布并附上时间戳

### 活动抽奖
1. 创建与会者姓名和电子邮件的 CSV
2. "从 attendees.csv 中随机选择 10 个获奖者"
3. 导出获奖者列表
4. 直接通过电子邮件联系获奖者

### 团队分配
1. 准备参与者列表
2. "将此列表随机分成 4 个相等的团队"
3. 查看分配
4. 分享团队名单

## 提示

- **记录过程**：保存时间戳和方法
- **公开宣布**：分享选择详情以确保透明度
- **检查资格**：验证获奖者是否符合竞赛规则
- **准备后备**：选择亚军以防获奖者不符合资格
- **导出结果**：保存获奖者列表作为记录
