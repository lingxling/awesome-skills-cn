# 域名头脑风暴器

此技能通过生成创意选项并检查实际可注册的域名，帮助您为项目找到完美的域名。

## 何时使用此技能

- 启动新项目或公司
- 推出产品或服务
- 创建个人品牌或作品集网站
- 重新品牌化现有项目
- 为副业项目注册域名
- 当首选域名被占用时寻找可用的替代方案

## 此技能的功能

1. **理解您的项目**：分析您正在构建的内容和目标受众
2. **生成创意名称**：创建相关、难忘的域名选项
3. **检查可用性**：验证多个 TLD 中哪些域名实际可用
4. **多个扩展名**：建议 .com、.io、.dev、.ai、.app 等
5. **提供替代方案**：如果首选被占用，提供变体
6. **品牌洞察**：解释为什么某些名称效果良好

## 域名命名提示

### 什么是好域名

✓ **简短**：理想情况下少于 15 个字符
✓ **易记**：易于回忆和拼写
✓ **可发音**：可以在对话中说出来
✓ **描述性**：暗示您的业务
✓ **品牌性**：足够独特以脱颖而出
✓ **无连字符**：口头分享更容易

### TLD 指南

- **.com**：通用，可信，适合企业
- **.io**：科技初创公司，开发者工具
- **.dev**：面向开发者的产品
- **.ai**：AI/ML 产品
- **.app**：移动或 Web 应用程序
- **.co**：.com 的替代品
- **.xyz**：现代，创意项目
- **.design**：创意/设计机构
- **.tech**：科技公司

## 示例工作流程

### 初创公司启动
1. 描述您的初创公司想法
2. 获取 10-15 个跨 TLD 的域名建议
3. 查看可用性和定价
4. 选择前 3 个收藏
5. 立即注册

### 个人品牌
1. 分享您的姓名和职业
2. 获取变体（firstname.com, firstnamelastname.dev 等）
3. 也检查社交媒体句柄可用性
4. 在平台上注册一致的品牌

### 产品命名
1. 描述产品和目标市场
2. 获取创意、可品牌化的名称
3. 检查商标冲突
4. 验证域名和社交可用性
5. 与目标受众测试名称

## 成功提示

1. **快速行动**：好域名很快被占用
2. **注册变体**：获取 .com 和 .io 以保护品牌
3. **避免数字**：口头交流困难
4. **检查社交媒体**：确保 @username 也可用
5. **大声说出来**：测试是否易于发音
6. **检查商标**：确保无法律冲突
7. **长远考虑**：5 年后是否仍然有意义？

## 定价上下文

在建议域名时，我会注意：
- 标准域名：~$10-15/年
- 高级 TLD（.io, .ai）：~$30-50/年
- 已占用域名：如果列出，市场价格
- 高级域名：数百到数千美元
