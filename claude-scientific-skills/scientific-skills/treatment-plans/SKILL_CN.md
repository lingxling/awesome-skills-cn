# 治疗计划编写

## 概述

治疗计划编写是通过循证干预、可测量目标和结构化随访来解决患者健康状况的临床护理策略的系统记录。此技能提供全面的LaTeX模板和验证工具，用于创建**简洁、重点突出**的治疗计划（标准3-4页），涵盖所有医学专科，完全符合监管要求。

**关键原则：**
1. **简洁且可操作**：治疗计划默认最多3-4页，仅关注影响护理决策的临床必要信息
2. **以患者为中心**：计划必须基于证据、可测量且符合医疗保健法规（HIPAA、文档标准）
3. **最小化引用**：仅在需要支持临床建议时使用简短的文本引用；避免冗长的参考书目

每个治疗计划应包括明确的目标、具体的干预措施、定义的时间表、监测参数和预期结果，这些都应与患者偏好和当前临床指南相一致 - 所有内容都以尽可能高效的方式呈现。

## 何时使用此技能

应在以下情况使用此技能：
- 为患者护理创建个性化治疗计划
- 记录慢性病管理的治疗干预措施
- 制定康复计划（物理治疗、职业治疗、心脏康复）
- 编写心理健康和精神科治疗计划
- 规划围手术期和外科护理路径
- 建立疼痛管理协议
- 使用SMART标准设定以患者为中心的目标
- 协调跨专科的多学科护理
- 确保治疗文档中的监管合规性
- 为医疗记录生成专业治疗计划

## 文档格式和最佳实践

### 文档长度选项

治疗计划根据临床复杂性和用例有三种格式选项：

#### 选项1：单页治疗计划（大多数情况下首选）

**使用场景**：简单的临床场景、标准协议、繁忙的临床环境

**格式**：单页包含所有必要的治疗信息，分为可扫描的部分
- 不需要目录
- 没有冗长的叙述
- 仅关注可操作项目
- 类似于精准肿瘤学报告或治疗建议卡

**必需部分**（全部在一页上）：
1. **标题框**：患者信息、诊断、日期、分子/风险概况（如适用）
2. **治疗方案**：特定干预措施的编号列表
3. **支持性护理**：简短的要点
4. **基本原理**：1-2句理由（标准方案可选）
5. **监测**：关键参数和频率
6. **证据水平**：指南参考或证据等级（例如，"1级，FDA批准"）
7. **预期结果**：时间表和成功指标

**设计原则**：
- 使用小框/表格进行组织（如临床治疗建议卡格式）
- 消除所有非必要文本
- 使用临床医生熟悉的缩写
- 密集信息布局 - 最大化每平方英寸的信息
- 考虑"快速参考卡"而非"全面文档"

**示例结构**：
```latex
[患者ID/诊断框在顶部]

目标患者人群
  患者数量、人口统计数据、关键特征

主要治疗方案
  • 药物1：剂量、频率、持续时间
  • 程序：具体细节
  • 监测：内容和时间

支持性护理
  • 关键支持性药物

基本原理
  简要临床理由

分子靶点/风险因素
  相关生物标志物或风险分层

证据水平
  指南参考、试验数据

监测要求
  关键实验室/生命体征、频率

预期临床获益
  主要终点、时间表
```

#### 选项2：标准3-4页格式

**使用场景**：中等复杂性、需要患者教育材料、多学科协调

采用Foundation Medicine首页摘要模型，外加2-3页详细内容。

#### 选项3：扩展5-6页格式

**使用场景**：复杂的合并症、研究方案、需要广泛的安全监测

### 首页摘要（Foundation Medicine模型）

**关键要求：所有治疗计划必须在第一页（仅第一页）上有完整的执行摘要，位于任何目录或详细部分之前。**

遵循Foundation Medicine的精准医学报告和临床摘要文档模型，治疗计划以一页执行摘要开始，提供对关键可操作信息的即时访问。整个摘要必须适合第一页。

**必需的首页结构（按顺序）：**

1. **标题和副标题**
   - 主标题：治疗计划类型（例如，"综合治疗计划"）
   - 副标题：特定条件或重点（例如，"2型糖尿病 - 年轻成人患者"）

2. **报告信息框**（使用`\begin{infobox}`或`\begin{patientinfo}`）
   - 报告类型/文档目的
   - 计划创建日期
   - 患者人口统计数据（年龄、性别、去识别化）
   - 主要诊断及ICD-10代码
   - 报告作者/诊所（如适用）
   - 使用的分析方法或框架

3. **关键发现或治疗亮点**（2-4个使用适当框类型的彩色框）
   - **主要治疗目标**（使用`\begin{goalbox}`）
     - 2-3个项目符号格式的SMART目标
   - **主要干预措施**（使用`\begin{keybox}`或`\begin{infobox}`）
     - 2-3个关键干预措施（药物、非药物、监测）
   - **关键决策点**（如紧急，使用`\begin{warningbox}`）
     - 重要的监测阈值或安全考虑
   - **时间表概述**（使用`\begin{infobox}`）
     - 简要治疗持续时间/阶段
     - 关键里程碑日期

**视觉格式要求：**
- 使用`\thispagestyle{empty}`从第一页移除页码
- 所有内容必须适合第1页（在`\newpage`之前）
- 使用彩色框（tcolorbox包），不同颜色用于不同信息类型
- 框应视觉突出且易于扫描
- 使用简洁的项目符号格式
- 目录（如包含）从第2页开始
- 详细部分从第3页开始

**示例首页结构：**
```latex
\maketitle
\thispagestyle{empty}

% 报告信息框
\begin{patientinfo}
  报告类型、日期、患者信息、诊断等
\end{patientinfo}

% 关键发现 #1：治疗目标
\begin{goalbox}[主要治疗目标]
  • 目标1
  • 目标2
  • 目标3
\end{goalbox}

% 关键发现 #2：主要干预措施
\begin{keybox}[核心干预措施]
  • 干预措施1
  • 干预措施2
  • 干预措施3
\end{keybox}

% 关键发现 #3：关键监测（如适用）
\begin{warningbox}[关键决策点]
  • 决策点1
  • 决策点2
\end{warningbox}

\newpage
\tableofcontents  % 目录在第2页
\newpage  % 详细内容从第3页开始
```

### 简洁文档

**关键：治疗计划必须优先考虑简洁性和临床相关性。默认最多3-4页，除非临床复杂性绝对需要更多细节。**

治疗计划应优先考虑**清晰度和可操作性**而非详尽细节：

- **重点突出**：仅包含影响护理决策的临床必要信息
- **可操作**：强调需要做什么、何时做以及为什么做
- **高效**：促进快速决策，同时不牺牲临床质量
- **目标长度选项**：
  - **1页格式**（简单病例首选）：包含所有必要信息的快速参考卡
  - **3-4页标准**：标准格式，包含首页摘要+支持性细节
  - **5-6页**（罕见）：仅适用于具有多种合并症或多学科干预的高度复杂病例

**简化指南：**
- **首页摘要**：使用单独的彩色框整合关键信息（目标、干预措施、决策点）- 这 alone 通常可以传达基本治疗计划
- **消除冗余**：如果信息在首页摘要中，不要在详细部分中逐字重复
- **患者教育部分**：仅包含3-5个关于关键主题和警告信号的要点
- **风险缓解部分**：仅突出关键药物安全问题和紧急行动（非详尽列表）
- **预期结果部分**：2-3个关于预期反应和时间表的简洁陈述
- **干预措施**：关注主要干预措施；次要/支持性措施以简短要点格式呈现
- **广泛使用表格和要点**以实现高效呈现
- **避免叙述性散文**，在结构化列表足够的情况下
- **在适当情况下合并相关部分**以减少页数

### 质量胜于数量

目标是专业、临床完整的文档，尊重临床医生的时间，同时确保全面的患者护理。每个部分都应增加价值；删除或浓缩不直接指导治疗决策的部分。

### 引用和证据支持

**使用最小、有针对性的引用来支持临床建议：**

- **首选文本引用**：使用简短的文本引用（作者年份）或简单参考，而非详尽的参考书目，除非特别要求
- **何时引用**：
  - 临床实践指南建议（例如，"根据ADA 2024指南"）
  - 特定药物剂量或方案（例如，"ACC/AHA建议"）
  - 需要证据支持的新颖或有争议的干预措施
  - 风险分层工具或经过验证的评估量表
- **何时不引用**：
  - 该领域广泛接受的标准护理干预措施
  - 基本医疗事实和常规临床实践
  - 一般患者教育内容
- **引用格式**：
  - 内联："开始使用二甲双胍作为一线治疗（ADA 2024年护理标准）"
  - 最小化："治疗遵循ACC/AHA心力衰竭指南"
  - 避免正式编号引用和详尽的参考书目部分，除非文档用于学术/研究目的
- **保持简洁**：3-4页的治疗计划最多应有0-3个引用，仅在对临床可信度或新颖建议必要时使用

## 核心功能

### 1. 一般医疗治疗计划

一般医疗治疗计划针对需要结构化治疗干预的常见慢性疾病和急性医疗问题。

#### 标准组件

**患者信息（去识别化）**
- 人口统计数据（年龄、性别、相关医疗背景）
- 活跃的医疗状况和合并症
- 当前药物和过敏
- 相关社会和家族史
- 功能状态和基线评估
- **HIPAA合规**：根据安全港方法移除所有18个标识符

**诊断和评估摘要**
- 主要诊断及ICD-10代码
- 次要诊断和合并症
- 严重程度分类和分期
- 功能限制和生活质量影响
- 风险分层（例如，心血管风险、跌倒风险）
- 预后指标

**治疗目标（SMART格式）**

短期目标（1-3个月）：
- **具体**：明确定义的结果（例如，"将HbA1c降至<7%"）
- **可测量**：可量化指标（例如，"将收缩压降低10 mmHg"）
- **可实现**：考虑患者能力的现实目标
- **相关**：与患者优先事项和价值观一致
- **有时限**：特定的时间框架（例如，"在8周内"）

长期目标（6-12个月）：
- 疾病控制或缓解目标
- 功能改善目标
- 生活质量提高
- 并发症预防
- 维持独立性

**干预措施**

*药物治疗*：
- 具有特定剂量、途径、频率的药物
- 滴定计划和目标剂量
- 药物相互作用考虑
- 不良反应监测
- 药物核对

*非药物治疗*：
- 生活方式修改（饮食、运动、戒烟）
- 行为干预
- 患者教育和自我管理
- 监测和自我跟踪（血糖、血压、体重）
- 辅助设备或适应性设备

*程序*：
- 计划的程序或干预措施
- 转诊至专科医生
- 诊断测试计划
- 预防保健（疫苗接种、筛查）

**时间表和计划**
- 具有特定时间框架的治疗阶段
- 预约频率（每周、每月、每季度）
- 里程碑评估和目标评估
- 药物调整计划
- 预期治疗持续时间

**监测参数**
- 要跟踪的临床结果（生命体征、实验室值、症状）
- 评估工具和量表（例如，PHQ-9、疼痛量表）
- 监测频率
- 干预或升级的阈值
- 患者报告的结果

**预期结果**
- 主要结果测量
- 成功标准和基准
- 预期改善时间表
- 治疗修改标准
- 长期预后

**随访计划**
- 预定的预约和重新评估
- 沟通计划（电话、安全消息传递）
- 紧急联系程序
- 紧急评估标准
- 过渡或出院计划

**患者教育**
- 对病情和治疗原理的理解
- 自我管理技能培训
- 药物管理和依从性
- 警告信号和何时寻求帮助
- 资源和支持服务

**风险缓解**
- 潜在不良反应和管理
- 药物相互作用和禁忌症
- 跌倒预防、感染预防
- 紧急行动计划
- 安全监测

#### 常见应用

- 糖尿病管理
- 高血压控制
- 心力衰竭治疗
- COPD管理
- 哮喘护理计划
- 高脂血症治疗
- 骨关节炎管理
- 慢性肾病

### 2. 康复治疗计划

康复计划专注于通过结构化治疗计划恢复功能、改善 mobility 和提高生活质量。

#### 核心组件

**功能评估**
- 基线功能状态（ADLs、IADLs）
- 活动范围、力量、平衡、耐力
- 步态分析和移动性评估
- 标准化测量（FIM、Barthel指数、Berg平衡量表）
- 环境评估（家庭安全、可访问性）

**康复目标**

*损伤级目标*：
- 将肩部屈曲改善至140度
- 股四头肌力量增加2/5 MMT等级
- 增强平衡（Berg评分>45/56）

*活动级目标*：
- 使用辅助设备独立行走150英尺
- 在扶手监督下爬12级楼梯
- 独立转移床到椅子

*参与级目标*：
- 重返工作岗位（带修改）
- 恢复娱乐活动
- 独立社区移动

**治疗干预措施**

*物理治疗*：
- 治疗性运动（加强、拉伸、耐力）
- 手动治疗技术
- 步态训练和平衡活动
- 物理因子（热、冰、电刺激、超声波）
- 辅助设备训练

*职业治疗*：
- ADL训练（洗澡、穿衣、梳理、进食）
- 上肢力量和协调
- 适应性设备和修改
- 能量节约技术
- 认知康复

*言语语言病理学*：
- 吞咽治疗和吞咽困难管理
- 沟通策略和辅助设备
- 认知语言治疗
- 语音治疗

*其他服务*：
- 娱乐治疗
- 水疗
- 心脏康复
- 肺康复
- 前庭康复

**治疗计划**
- 频率：3次/周PT，2次/周OT（示例）
- 会话持续时间：45-60分钟
- 治疗阶段持续时间（急性、亚急性、维持）
- 预期总持续时间：8-12周
- 重新评估间隔

**进展监测**
- 每周功能评估
- 标准化结果测量
- 目标达成量表
- 疼痛和症状跟踪
- 患者满意度

**家庭锻炼计划**
- 具体练习，包括重复/组数/频率
- 注意事项和安全说明
- 进展标准
- 自我监测策略

#### 专科康复

- 卒中后康复
- 骨科康复（关节置换、骨折）
- 心脏康复（MI后、手术后）
- 肺康复
- 前庭康复
- 神经康复
- 运动损伤康复

### 3. 心理健康治疗计划

心理健康治疗计划通过综合心理治疗、药物治疗和心理社会干预来解决精神疾病。

#### 基本组件

**精神科评估**
- 主要精神科诊断（DSM-5标准）
- 症状严重程度和功能障碍
- 共存的心理健康状况
- 物质使用评估
- 自杀/杀人风险评估
- 创伤史和PTSD筛查
- 心理健康的社会决定因素

**治疗目标**

*症状减少*：
- 降低抑郁严重程度（PHQ-9评分从18降至<10）
- 减少焦虑症状（GAD-7评分<5）
- 改善睡眠质量（匹兹堡睡眠质量指数）
- 稳定情绪（减少情绪发作）

*功能改善*：
- 重返工作或学校
- 改善社会关系和支持
- 增强应对技能和情绪调节
- 增加参与有意义活动

*恢复导向目标*：
- 建立韧性和自我效能
- 发展危机管理技能
- 建立可持续的健康常规
- 实现个人恢复目标

**治疗干预措施**

*心理治疗*：
- 循证模式（CBT、DBT、ACT、心理动力学、IPT）
- 会话频率（每周、每两周）
- 治疗持续时间（12-16周，持续）
- 具体技术和目标
- 团体治疗参与

*精神药理学*：
- 药物类别和原理
- 起始剂量和滴定计划
- 目标症状
- 预期反应时间表（抗抑郁药为2-4周）
- 副作用监测
- 联合治疗考虑

*心理社会干预*：
- 病例管理服务
- 同伴支持计划
- 家庭治疗或心理教育
- 职业康复
- 支持性住房或社区融合
- 物质滥用治疗

**安全计划**
- 危机联系人和紧急服务
- 警告信号和触发因素
- 应对策略和自我安慰技术
- 安全环境修改
- 手段限制（枪支、药物）
- 支持系统激活

**监测和评估**
- 症状评分量表（每周或每两周）
- 药物依从性和副作用
- 自杀意念筛查
- 功能状态评估
- 治疗参与和治疗联盟

**患者和家庭教育**
- 关于诊断的心理教育
- 治疗原理和期望
- 药物信息
- 复发预防策略
- 社区资源

#### 心理健康状况

- 重度抑郁症
- 焦虑障碍（GAD、恐慌、社交焦虑）
- 双相情感障碍
- 精神分裂症和精神障碍
- PTSD和创伤相关障碍
- 进食障碍
- 物质使用障碍
- 人格障碍

### 4. 慢性病管理计划

针对需要持续监测、治疗调整和多学科协调的慢性疾病的综合长期护理计划。

#### 关键特点

**疾病特定目标**
- 基于指南的循证治疗目标
- 适合阶段的干预措施
- 并发症预防策略
- 疾病进展监测

**自我管理支持**
- 患者激活和参与
- 共享决策
- 症状变化的行动计划
- 技术支持的监测（应用程序、远程监测）

**护理协调**
- 初级保健医生监督
- 专科咨询和共同管理
- 护理过渡（医院到家庭）
- 跨提供者的药物管理
- 沟通协议

**人群健康整合**

- 预防保健和筛查计划
- 质量措施报告
- 护理差距识别

#### 适用条件

- 1型和2型糖尿病
- 心血管疾病（CHF、CAD）

### 5. 围手术期护理计划

针对外科和手术患者的结构化计划，涵盖术前准备、术中管理和术后恢复。

#### 组件

**术前评估**
- 手术适应症和计划程序
- 术前风险分层（ASA分级、心脏风险）
- 医疗状况优化
- 药物管理（继续、停用）
- 术前测试和许可
- 知情同意和患者教育

**围手术期干预措施**
- 增强术后恢复（ERAS）协议
- 静脉血栓栓塞预防
- 抗生素预防
- 血糖控制策略
- 疼痛管理计划（多模式镇痛）

**术后护理**
- 立即恢复目标（24-48小时）
- 早期活动协议
- 饮食进展
- 伤口护理和引流管理
- 疼痛控制方案
- 并发症监测

**出院计划**
- 活动限制和进展
- 药物核对
- 随访预约
- 家庭健康或康复服务
- 重返工作时间表

### 6. 疼痛管理计划

使用循证干预和阿片类药物节约策略的急性和慢性疼痛多模式方法。

#### 综合组件

**疼痛评估**
- 疼痛位置、性质、强度（0-10量表）
- 时间模式（持续、间歇性、突破性）
- 加重和缓解因素
- 功能影响（睡眠、活动、情绪）
- 以前的治疗和反应
- 心理社会因素

**多模式干预**

*药物治疗*：
- 非阿片类镇痛药（对乙酰氨基酚、NSAIDs）
- 辅助药物（抗抑郁药、抗惊厥药、肌肉松弛剂）
- 局部药物（利多卡因、辣椒素、双氯芬酸）
- 阿片类药物治疗（适当时，带风险缓解）
- 滴定和轮换策略

*介入程序*：
- 神经阻滞和注射
- 射频消融
- 脊髓刺激
- 鞘内药物递送

*非药物治疗*：
- 物理治疗和运动
- 疼痛认知行为治疗
- 正念和放松技术
- 针灸
- TENS装置

**阿片类药物安全（当处方时）**
- 适应症和计划持续时间
- 处方药监测计划（PDMP）检查
- 阿片类药物风险评估工具
- 纳洛酮处方
- 治疗协议
- 随机尿药物筛查
- 频繁随访和重新评估

**功能目标**
- 特定活动改善
- 睡眠质量提高
- 疼痛干扰减少
- 生活质量改善
- 重返工作或有意义的活动

## 最佳实践

### 简洁和重点（最高优先级）

**治疗计划必须简洁并专注于可操作的临床信息：**

- **1页格式是首选**：对于大多数临床场景，单页治疗计划（如精准肿瘤学报告）提供所有必要信息
- **默认使用最短格式**：从1页开始；仅在临床复杂性真正需要时扩展
- **每个句子必须增加价值**：如果一个部分不改变临床决策，完全省略它
- **考虑"快速参考卡"而非"全面教科书"**：忙碌的临床医生需要可扫描、密集的信息
- **避免学术冗长**：这是临床文档，不是文献综述或教学文档
- **按复杂性划分的最大长度**：
  - 简单/标准病例：1页
  - 中等复杂性：3-4页（首页摘要+详细信息）
  - 高复杂性（罕见）：最多5-6页

### 首页摘要（最重要）

**始终创建一页执行摘要作为第一页：**
- 第一页必须仅包含：标题、报告信息框和关键发现框
- 这提供了类似于精准医学报告的一目了然的概述
- 目录和详细部分从第2页或之后开始
- 将其视为忙碌临床医生可以在30秒内扫描的"临床亮点"页面
- 为不同的关键发现（目标、干预措施、决策点）使用2-4个彩色框
- **一个强有力的首页通常可以独立存在** - 后续页面用于详细信息，而非重复

### SMART目标设定

所有治疗目标应符合SMART标准：

- **具体**："将HbA1c降至<7%"而非"更好的糖尿病控制"
- **可测量**：使用可量化指标、经过验证的量表、客观测量
- **可实现**：考虑患者能力、资源、社会支持
- **相关**：与患者价值观、优先事项和生活环境一致
- **有时限**：为目标实现和重新评估定义明确的时间框架

### 以患者为中心的护理

✓ **共享决策**：让患者参与目标设定和治疗选择  
✓ **文化能力**：尊重文化信仰、语言偏好、健康素养  
✓ **患者偏好**：尊重治疗偏好和个人价值观  
✓ **个性化**：根据患者的独特情况定制计划  
✓ **赋权**：支持患者激活和自我管理  

### 循证实践

✓ **质量措施**：纳入HEDIS、CMS质量措施  
✓ **比较有效性**：使用已证明有效的治疗  
✓ **避免低价值护理**：消除不必要的测试和干预

### 文档标准

## LaTeX模板使用

### 模板选择

根据临床背景和所需长度选择适当的模板：

#### 简洁模板（首选）

1. **one_page_treatment_plan.tex** - **大多数情况下的首选**
   - 所有临床专科
   - 标准协议和简单病例
   - 类似于精准肿瘤学报告的快速参考格式
   - 密集、可扫描、以临床医生为中心
   - 除非复杂性要求更多细节，否则使用此模板

#### 标准模板（3-4页）

仅在单页格式因复杂性不足时使用：

2. **general_medical_treatment_plan.tex** - 初级保健、慢性病、一般医学

## 常见用例

### 示例1：2型糖尿病管理

**场景**：58岁新诊断2型糖尿病患者，HbA1c 8.5%，BMI 32

**模板**：`general_medical_treatment_plan.tex`

**目标**：
- 短期：3个月内将HbA1c降至<7.5%
- 长期：6个月内达到HbA1c <7%，减重15磅

**干预措施**：
- 药物治疗：二甲双胍500mg BID，滴定至1000mg BID
- 生活方式：地中海饮食，每周150分钟中等运动
- 教育：糖尿病自我管理教育，血糖监测

### 示例2：卒中后康复

**场景**：70岁左侧MCA卒中患者，右侧偏瘫

**模板**：`rehabilitation_treatment_plan.tex`

**目标**：
- 短期：4周内将右臂力量从2/5改善至3/5
- 长期：12周内使用手杖独立行走150英尺

**干预措施**：
- PT 3次/周：步态训练、平衡、加强
- OT 3次/周：ADL训练、上肢功能
- SLP 2次/周：吞咽治疗

### 示例3：重度抑郁症

**场景**：35岁中度抑郁症患者，PHQ-9评分16

**模板**：`mental_health_treatment_plan.tex`

**目标**：
- 短期：8周内将PHQ-9降至<10
- 长期：达到缓解（PHQ-9 <5），重返工作

**干预措施**：
- 心理治疗：每周CBT会话
- 药物：舍曲林50mg每日，滴定至100mg
- 生活方式：睡眠卫生，每周5次30分钟运动

### 示例4：全膝关节置换术

**场景**：68岁因骨关节炎计划进行右侧TKA

**模板**：`perioperative_care_plan.tex`

**术前目标**：
- 优化糖尿病控制（血糖<180）
- 按协议停用抗凝药物
- 完成医疗许可

**术后目标**：
- POD 1前行走50英尺
- POD 3前膝关节屈曲90度
- POD 2-3前出院回家，带PT服务

### 示例5：慢性下背痛

**场景**：45岁慢性非特异性下背痛患者，疼痛7/10

**模板**：`pain_management_plan.tex`

**目标**：
- 短期：6周内将疼痛降至4/10
- 长期：全职重返工作，疼痛2-3/10

**干预措施**：
- 药物治疗：加巴喷丁300mg TID，度洛西汀60mg每日
- PT：核心加强，麦肯齐练习2次/周×8周
- 行为：疼痛CBT，正念冥想
- 介入：如反应不足，考虑腰椎ESI

- AOTA实践指南

## 专业文档样式

### 概述

治疗计划可以使用`medical_treatment_plan.sty` LaTeX包进行专业医疗文档样式增强。此自定义样式将普通学术文档转换为视觉吸引力强、颜色编码的临床文档，在保持科学严谨性的同时提高可读性和可用性。

### 医疗治疗计划样式包

`medical_treatment_plan.sty`包（位于`assets/medical_treatment_plan.sty`）提供：

**专业配色方案**
- **主蓝色**（RGB：0, 102, 153）：标题、章节标题、主要强调
- **次蓝色**（RGB：102, 178, 204）：浅色背景、微妙强调
- **强调蓝色**（RGB：0, 153, 204）：超链接、关键亮点
- **成功绿色**（RGB：0, 153, 76）：目标、积极结果
- **警告红色**（RGB：204, 0, 0）：警告、关键信息
- **深灰色**（RGB：64, 64, 64）：正文
- **浅灰色**（RGB：245, 245, 245）：背景填充

**样式元素**
- 带有专业规则的自定义彩色页眉和页脚
- 带有下划线的蓝色章节标题，层次清晰
- 带有彩色标题和交替行的增强表格格式
- 带有彩色项目符号和编号的优化列表间距
- 具有适当边距的专业页面布局

### 自定义信息框

样式包包括五个专门的框环境，用于组织临床信息：

#### 1. 信息框（蓝色边框，浅灰色背景）

用于一般信息、临床评估和测试计划：

```latex
\begin{infobox}[标题]
  \textbf{关键信息：}
  \begin{itemize}
    \item 临床评估详情
    \item 测试计划
    \item 一般指导
  \end{itemize}
\end{infobox}
```

**使用场景**：代谢状态、基线评估、监测计划、滴定方案

#### 2. 警告框（红色边框，黄色背景）

用于关键决策点、安全协议和警报：

```latex
\begin{warningbox}[警报标题]
  \textbf{重要安全信息：}
  \begin{itemize}
    \item 关键药物相互作用
    \item 安全监测要求
    \item 需要立即行动的红旗症状
  \end{itemize}
\end{warningbox}
```

**使用场景**：药物安全、决策点、禁忌症、紧急协议

#### 3. 目标框（绿色边框，绿色 tint背景）

用于治疗目标、目标和成功标准：

```latex
\begin{goalbox}[治疗目标]
  \textbf{主要目标：}
  \begin{itemize}
    \item 3个月内将HbA1c降至<7\%
    \item 12周内实现5-7\%体重减轻
    \item 完成糖尿病教育计划
  \end{itemize}
\end{goalbox}
```

**使用场景**：SMART目标、目标结果、成功指标、CGM目标

#### 4. 关键点框（蓝色背景）

用于执行摘要、关键要点和重要建议：

```latex
\begin{keybox}[关键亮点]
  \textbf{基本要点：}
  \begin{itemize}
    \item 主要治疗方法
    \item 关键患者指导
    \item 优先干预措施
  \end{itemize}
\end{keybox}
```

**使用场景**：计划概述、板块方法指导、重要饮食指南

#### 5. 紧急框（大红色设计）

用于紧急联系人和紧急协议：

```latex
\begin{emergencybox}
  \begin{itemize}
    \item \textbf{紧急服务：} 911
    \item \textbf{内分泌科办公室：} [电话]（工作时间）
    \item \textbf{下班后热线：} [电话]（晚上/周末）
    \item \textbf{药房：} [电话和位置]
  \end{itemize}
\end{emergencybox}
```

**使用场景**：紧急联系、关键热线、紧急资源信息

#### 6. 患者信息框（白色带蓝色边框）

用于患者人口统计和基线信息：

```latex
\begin{patientinfo}
  \begin{tabular}{ll}
    \textbf{年龄：} & 23岁 \\
    \textbf{性别：} & 男 \\
    \textbf{诊断：} & 2型糖尿病 \\
    \textbf{计划开始日期：} & \today \\
  \end{tabular}
\end{patientinfo}
```

**使用场景**：患者信息部分、人口统计数据

### 专业表格格式

带有医疗样式的增强表格环境：


- 专业边框和填充

### 样式文档的最佳实践

1. **适当的框使用**
   - 将框类型与内容目的匹配（目标→绿色，警告→黄色/红色）
   - 不要过度使用框；仅用于真正重要的信息
   - 保持框内容简洁和重点突出

2. **视觉层次**
   - 使用章节样式进行结构
   - 框用于强调和组织
   - 表格用于比较数据
   - 列表用于顺序或分组项目

3. **颜色一致性**
   - 坚持定义的配色方案
   - 使用`\textcolor{primaryblue}{\textbf{文本}}`进行强调
   - 保持一致的含义（红色=警告，绿色=目标）

4. **白色空间**
   - 不要用框过度拥挤页面
   - 在主要部分之间使用`\vspace{0.5cm}`
   - 在彩色元素周围留出呼吸空间

5. **专业外观**
   - 保持可读性为首要任务
   - 确保足够的对比度以提高可访问性
   - 测试灰度打印输出
   - 保持整个文档的样式一致

6. **表格格式**
   - 对所有标题行使用`\tableheadercolor`
   - 对>3行的表格应用`\tablerowcolor`到交替行
   - 保持列宽平衡
   - 对大型表格使用`\small\sffamily`
